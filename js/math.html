Cálculos

Métodos para realizar cálculos matemáticos.

Math()

Permite realizar cálculos dándole como parámetros una operación matemática. Devuelve el resultado de la operación.

Acompañado de otras palabras reservadas puede hacer cálculos predefinidos.

Convertir números en números enteros

  • Math.round(x) : Devuelve el argumento redondeado a su número entero más próximo.

  • Math.ceil(x) : Devuelve el argumento redondeado al alza.

  • Math.floor(x) : Devuelve el argumento redondeado a la baja.

  • Math.trunc(x) : Devuelve sólo la parte del número entero, sin decimales.

Métodos .Math()

Math.sign(x)

Devuelve si el argumento es un número negativo (-1), null(0) o positivo(1).

Math.pow(x, y)

Eleva a la potencia y sobre x. x elevado a y.

Math.sqrt(x)

Devuelve la raíz cuadrada el argumento.

Math.abs(x)

Convierte un número negativo en positivo.

Math.sin()

Devuelve el seno( valor entre -1 y 1) de un ángulo dado como argumento en radianes.

Math.cos()

Devuelve el coseno (valor entre -1 y 1 ) de un angulo dado como argumento en radianes.

Math.min() Math.max()

Devuelve el valor más alto o el más bajo de una lista de argumentos.

Math.random()

Devuelve un número al azar entre 0 (incluído) y 1 ( no incluído).

Math.log() Math.log2() Math.log10()

Devuelve el logaritmo natural e del argumento. Se puede añadir número después de log para cambiar la base del logaritmo.
