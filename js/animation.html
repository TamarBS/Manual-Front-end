Animaciones y transiciones

Cambiando los estilos CSS de los elementos HTML es posible crear la ilusión de cambiar y dar movimiento a los contenidos de la web.

Todas las animaciones deben estar contenidas en un contenedor padre. Ese contenedor padre tiene que tener position relativa y el hijo position absolute.

let id = setInterval( frame, 5);

if( condiciones para terminar la animación ) {
clearInterval( id );
