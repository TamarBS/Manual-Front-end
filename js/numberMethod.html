Métodos para números

Métodos para utilizar con números.

Métodos básicos

Pueden ser usados en cualquier número.

toString()

Cambia el tipo de número a string. Si se añade un parámetro puede hacer que cambie la base del número. Ej. Cambiar 7 a base binaria toString(2) = 111

toExponential()

Devuelve un string con un número usando anotación exponencial. Ej.: 9.236 toExponential(2) = 9.236e+0

toFixed()

Devuelve un string con el número y la cantidad de decimales especificada en el parámetro. toFixed(2) es perfecta para trabajar con dinero.

toPrecision()

Devuelve un string con la cifra y tantos números como se haya pasado en el parámetro.

valueOf()

Convierte los números de tipo objeto en tipo número.

Existen otros tres métodos que convierten variables en números: Number(), parseFloat() y parseInt().

Métodos estáticos

Sólo pueden ser usados en "Number"

Number() convierte fechas a número. Ej.: Number(new Date("1970+01+01")); El método Date devuelve el número de milisegundos desde el 1 de Enero de 1970.

Number.isFinite()

Devuelve true si el argumento es un número finito.

Number.isInteger()

Devuelve true si el argumento es un número entero.

Number.isNan()

Devuelve true si el argumento no es NaN.

Number.isSafeInteger()

Un safe integer es un número entero que no excede las 15 cifras y está entre -9007199254740991 y 9007199254740991. Este método devuelve true o false dependiendo de si el número es un íntegro seguro o no.

Number.parseInt()

Analiza un string y devuelve un número entero. Si el número no se puede convertir el valor será NaN.

Number.parseFloat()

Analiza un string y devuelve un número. Admite espacios pero cojerá sólo el primer número. Si no encuentra número será NaN.

Propiedades de los números

EPSILON

Representa la diferencia más pequeña entre 1 y el siguiente número de punto flotante mayor que 1. Los número de punto flotante tienen precisión limitada. Significa que no todos los números pueden ser representados con exactitud en memoria.

MAX_VALUE

Representa el máximo valor posible en JavaScript.

MIN_VALUE

Representa el mínimo valor posible en JavaScript.

MAX_SAFE_INTEGER

Representa el máximo valor entero posible en JavaScript.

MIN_SAFE_INTEGER

Representa el mínimo valor entero posible en JavaScript.

POSITIVE_INFINITY

Devuelve Infinito

NEGATIVE_INFINITY

Devuelve -infinito
