Volver a JavaScript

Escuchadores

Se denominan Event Handlers o Event Listeners y sirven para manejar y verificar inputs, acciones del usuario o del navegador.

onchange

Un elemento HTML ha sido cambiado.

onclick

El usuario a clickado un elemento HTML.

onmouseover

El usuario ha colocado el ratón por encima de un elemento HTML.

onmouseout

El usuario a sacado el ratón de encima de un elemento HTML.

onkeydown

El usuario a pulsado alguna tecla del teclado.

onload

El navegador a terminado de cargar la página.
