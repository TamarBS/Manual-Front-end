Métodos para cadenas de texto

Métodos para utilizar con cadenas de texto.

Buscar en el string
indexOf()

Devuelve la posición de la primera vez que encuentra el texto buscado. Si no encuentra el texto devuelve -1.

Acepta otro parámetro para la posición desde dónde empezar a buscar. Busca de derecha a izquierda.

No puede hacer búsquedas con expresiones regulares.

let text = "hola, hola, holita";
let index = text. indexOf( "hola" );
// Resultado: 0. Encuentra el primer hola en primera posición.
lastIndexOf()

Devuelve la posición de la última palabra que coincide con la que se está buscando. Si no encuentra el texto devuelve -1.

Acepta otro parámetro para la posición desde dónde empezar a buscar. Busca de izquierda a derecha.

let text = "hola, hola, holita";
let index = text. lastIndexOf( "hola" );
// Resultado: 6. Encuentra el último hola.
search()

Busca la cadena de texto como texto o como expresión regular y devuelve su posición.

No acepta un segundo parámetro como indexOf().

let text = "hola, hola, holita";
let index = text. search( "hola" );
// Resultado: 0. Encuentra el primer hola en primera posición.
match()

Devuelve un array con los resultados que coincidan de comparar una cadena de texto con otra o una cadena de texto con una expresión regular.

Comparando cadenas de texto se parará con la primera coincidencia, si se le añade una expresión regular con / /gi que aporta búsqueda global, es decir, todas las que haya, y la i que hace que no discrimine mayúsuclas, creará un array con todas las coincidencias.

let texto = "Hola, holita, hola";
let array = texto. match( /hol/gi );
document.getElementById("...").innerHTML = array; // Resultado: Hol, hol, hol
matchAll()

Cómo match() pero devuelve el resultado de una iteración, que le permite devolver todas las coincidencias y no sólo quedarse con la primera.

string. matchAll( "string a buscar" );
Array. from( nombreArray );

let texto = "Hola, holita, hola";
let array = texto. matchAll( "hol" );
document.getElementById("...").innerHTML = Array.from(array); // Resultado: hol, hol
includes()

Devuelve true si el valor está en la cadena de texto.

let texto = "Hola, holita";
texto.includes("hol");
// Resultado: true

Permite añadir un segundo parámetro para comenzar la búsqueda a partir de esa posición.

startsWith() endsWith()

Devuelve verdadero si encuentra la búsqueda al principio (startsWith()) o al final (endsWith()).

Aceptan un segundo parámetro para empezar la búsqueda desde una posición concreta.

Controlar símbolos
isWellFormed()

sirve para saber si un código del string da error o no. Si da error el resultado es false.

toWellFormed()

devuelve un nuevo string dónde los códigos que dan error en isWellFormed() se reemplazan por un caracter unicode.

Eliminar espacios en blanco
trim()

Elimina espacios en blanco de ambos lados del string.

trimStart()

Sólo elimina espacios en blanco de la parte inicial de un string.

trimEnd()

Sólo elimina espacios en blanco de la parte final de un string.

Extraer caracteres
charAt(position)

permite extraer caracteres de un string.

let text = "Hola";
let caracter = text.charAt(0);
// Resultado: H
charCodeAt(position) y codePointAt():

permite extraer caracteres de un string pero devuelve el caracter en código UTF-16 que es un número entre 0 y 65535. codePointAt() puede devolver valores por encima de 65535.

let text = "Hola";
let caracter = text.charCodeAt(0);
// Resultado: 72
at(position) y [ ]

permite extraer caracteres de un string. at() permite acceder utilizando de posición del index números en negativo. [] puede devolver resultados inesperados, por ejemplo si no encuentra caracter devolverá "undefined" dónde charAt() devolvería un string vacío.

let text = "Hola";
let caracter = text.at(0);
let caracter = text[0];
// Resultado para ambos: H

text[0] = "A"; // Es read only, por lo que no funciona y no aplicará ese cambio. Aunque no dará error en consola.
slice(start, end)

extraen parte de un string Si sólo se define el primer parámetro se extraerá el resto del string desde esa posición. Si el parámetro es un número negativo la primera prosición se cuénta de atrás hacia delante..

let text = "12345 manzana";
let part = text.slice(7, 13);
// Resultado: manzana
substring(start, end)

extraen parte de un string. Si sólo se define el primer parámetro se extraerá el resto del string desde esa posición. No funciona con parámetros en negativo.

let text = "12345 manzana";
let part = text.substring(7, 13);
// Resultado: manzana
Largura en caracteres
length

Hace varias funciones según el tipo de objeto.

  • string : Cuenta el número de caracteres de un string.

  • array :Cuénta el número de elementos que contiene el array.

let text = "Hola";
let lenght = text.length;
// Resultado: 4
padStart()

Rellenan con un valor dado un string hasta que cumpla con la longitud de caracteres especificada.

let text = "5";
let padded = text.padStart(4, "0");
// Resultado: 0005 Le añade el 0 tantas veces sea necesario para conseguir que su length sea de 4 caracteres.
padEnd()

Rellenan con un valor dado un string hasta que cumpla con la longitud de caracteres especificada.

let text = "5";
let padded = text.padEnd(4, "0");
// Resultado: 5000 Le añade el 0 tantas veces sea necesario para conseguir que su length sea de 4 caracteres.
Mayúsculas y minúsculas
toUpperCase()

cambia todos los caracteres a mayúsculas.

toLoswerCase()

cambia todos los caracteres a minúsculas.

repeat()

crea un nuevo string que contiene tantas repeticiones como se haya especificado en el parámetro.

Reemplazar partes del texto
replace()

Reemplaza la parte de un string por otro texto.Se le pasan dos parámetros, primero la parte del string que se quiere cambiar y después la sustitución.

let text = "Hola holita";
let newText = text.replace("holita", "qué tal?);
// Sólo cambia el término una vez, si hay varios términos iguales sólo aplicará el método en el primero.
// Para cambiar todas las palabras que coincidan se añade /g al término a cambiar de la siguiente manera:
let newText = text.replace(/holita/g, "qué tal?);
Para que no discrimine mayúsculas de minúsculas se puede usar /i. Ej.: /holita/i
replaceAll()

Permite reemplazar todas las palabras de un string. Hace como la opción de la expresión regular /holita/g.

text = text.replaceAll("cats", "dogs");
split()

convierte strings en arrays. Cómo parámetro se define el separador a omitir. Si por ejemplo mi string es 1, 2, 3, 4 pondré de parámetro "," para que mi array no sea 1, ,, 2, ,, 3, ,, 4. porque el array usa comas para separar valores así que tomará como valor la coma del string.

toString()

convierte números en string.

Unir

concat() : une dos o más strings. Hace lo mismo que el operador +.

let text1 = "Hola";
let text2 = "Holita";
let text3 = text1.concat(" ", text2);
// Resultado: Hola Holita
