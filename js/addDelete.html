Añadir o eliminar Elementos HTML

Permite crear nuevos elementos HTML o eliminar los que ya existen.

Crear elementos HTML

  • document.createElement(element) Crea un nuevo elemento HTML.

  • document.appendChild(element) Añade al final un nuevo hijo.

createElement(element)

Crea un nuevo elemento HTML.

Eliminar elementos HTML

  • document.removeChild(element) Elimina un elemento HTML.

  • document.replaceChild(new, old) Sustituye un elemento HTML por otro.
