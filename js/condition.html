Condicionales

Para los casos en los que se quiere que una acción se lleve a cabo sólo cuándo se cumplan ciertas condiciones existen los siguientes estados condicionales (conditional statements).

Estados condicionales

if

Para ejecutar una acción si una condición se cumple.

if (condición) {
// código para ser ejecutado cuándo la condición se cumple
}
else

Para ejecutar una acción si una condición no se cumple.

if (condición) {
// código para ser ejecutado cuándo la condición se cumple
} else {
// código para ser ejecutado si la condición no se ha cumplido
}
else if

Para ejecutar una acción si una segunda condición dentro de una primera se cumple después de que la primera no se haya cumplido.

if (condición) {
// código para ser ejecutado cuándo la condición se cumple
} else if (conditción 2) {
// código para ser ejecutado si la condición del if no se ha cumplido pero sí que se cumple la condición 2 del else if
} else {
// código para ser ejecutado si ninguna condición se cumple
}
switch

Permite ejecutar diferentes acciones basado en diferentes condiciones. Se ejecutará el código del caso que sea igual en valor y tipo al resultado de la expresión.

La expresión es el valor que se quiere comparar con los diferentes casos. Si el resultado de la expresión puede no coincidir con alguno de los casos se utiliza el default que será su código el que se ejecutará en caso de no haber tenido la expresión ninguna coincidencia con los casos.

switch(expresión) {
case x:
// código que se ejecutará si el valor de la expresión es el mismo que la x de este caso.
break;
case y:
// código que se ejecutará si el valor de la expresión es el mismo que la y de este caso.
break;
default:
// código que se ejecutará si el valor de la expresión no coincide ni con x ni con y.
}

Si se quiere que más de un caso ejecute un código se pueden poner los dos casos seguidos que hará que compartan código.

switch(expresión) {
case x:
case y:
// código que se ejecutará si el valor de la expresión es el mismo que la x o la y de este caso.
break;
case z:
// código que se ejecutará si el valor de la expresión es el mismo que la z de este caso.
break;
}

Bucles condicionales

Existen dos palabras reservadas para parar o saltar iteraciones en los bucles.

  • break : Para el bucle aunque no haya llegado a cumplir con todas las posibilidades de cumplir con la condición.

  • continue : Se salta la repetición en la vez que le toca pero luego continúa con el bucle hasta el final.

Estas palabras reservadas no sólo se pueden usar dentro de iteraciones, se pueden usar para parar bloques de código en algún punto, pero parar eso tienen que tener vinculado un "label". Un label es una palabra para designar un bloque de código.

const cars = ["BMW", "Volvo", "Saab", "Ford"];
list: {
// Este es el label. Sintaxis label: { ... }
text += cars[0] + " ";
text += cars[1] + " ";
break list;
// El break o continue tiene que estar vinculado al label: Sitaxis break|continue labelName; text += cars[2] + " ";
text += cars[3] + " ";
}
// Este bloque de código sólo se ejecutará hasta cars[1]. Resultado: BMW, Volvo
for

Ejecuta un bucle que revisa una condición en cada vuelta mientras se incrementa el valor de la expresión (valor a comparar). En algún momento la condición se deja de cumplir y en ese momento detiene el bucle.

for (let i = 0; i < cars.length; i++) {
// for (declarar variable (se ejecuta una vez); condición; incremento|descuento(se ejecuta cada vez que se cumple la condición.)) {
// Las tres expresiones del for son opcionales. La variable se puede declarar antes del for. La condición se puede obviar también, pero será necesario añadir un break dentro del bucle. De lo contrario hará un bucle infinito y romperá el sitio web. El incremento o descuento se puede omitir también mientras lo incluyas como código dentro del bucle después de la acción. De todas formas es necesario incluir los punto y coma ;. Quedaría algo así: for (; condición ;) {...}
text += cars[i] + "
";
}
for/in

Ejecuta un bucle en las propiedades de un objeto.

for (key in object) {
// ejecuta el código mientras encuentra propiedades en el objeto.
}
const person = {fname:"John", lname:"Doe", age:25};
let text = "";
for (let x in person) {
text += person[x];
}
for/of

Ejecuta un bucle en los valores de un objeto iterable.

for (variable of iterable) {
// code block to be executed
}

Los objetos iterables puede ser arrays, strings, Maps y NodeLists entre otros

const cars = ["BMW", "Volvo", "Mini"];
let text = "";
for (let x of cars) {
text += x;
}
while

Ejecuta un bucle mientras una condición se cumpla. Es como un for al que se le ha omitido la variable y el incremento sin necesidad de añadir el punto y coma en los parámetros.

while (condition) {
// code block to be executed
}
while (i < 10) {
text += "The number is " + i;
i++;
}
// Si no se añade el incremento o decremento se puede generar un loop infinito que romperá la web.
do/while

Primero hace una primera acción y después revisa si una condición se cumple para repetir el bucle o parar.

do {
// code block to be executed
}
while (condition);
