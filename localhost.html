localhost

Es una dirección de red especial que se refiere a tu propio ordenador local. Permite al navegador comuniarse con software y aplicaciones que se ejecutan localmente.

¿Cómo funciona?

Funciona mediante el uso de la dirección IP especial de loopback. Un loopback se refiere a una dirección o interfaz virtual que permite que un dispositivo se comunique consigo mismo.

Mediante las palabras localhost en el navegador, entiende que tiene que contectarse con la IP del ordenador y esto le permite acceder a los archivos locales de forma segura sin necesidad de internet.

En desarrollo se suele configurar un servidor web local en la máquina, como Apache, Nginx o específicos para el lenguaje de programación utilizado.

¿Cómo utilizarlo?

Instalar un servidor web.

Apache, Nginx, XAMPP, MAMP, WAMP. Yo he trabajado con MAMP (el elefantito).

Configurar el MAMP.

Encender el servidor en el botón on-off.

MAMP>preferencias>Server>Document root.

Añadir la ruta al proyecto que se quiere ver en localhost.

Aceptar.

Abrir en el navegador.

En la url escribir localhost o también servirá escribir "127.0.0.1".

Con esto ya se debería estar viendo el proyecto en localhost.

Cerrar al terminar.

No dejar abierto. Pulsa de nuevo el botón on-off para apagar el MAMP.
