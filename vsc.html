Visual Studio Code

Lista de acciones útiles que se pueden utilizar en este editor de código.

Atajos de teclado

Ajustar ancho de línea al ancho de pantalla

ALT + z

Cambiar varias líneas similares a la vez:

Alt + Shift + i

Copiar línea encima o debajo:

Alt + Shift + arriba/abajo

Crear etiquetas automáticamente

"etiqueta" + "." Crea una etiqueta con sus atributos vacíos. Ej.: div.

"etiqueta" + "*" + "número" Crea tantas etiquetas como número se haya elegido. Ej.: div*3

Escribir en varios sitios a la vez:

Alt + mouse click

Escribir en varios sitios a la vez seleccionado el inmediato superior/posterior:

Alt + Control + arriba/abajo

Escribir estructura de HTML automática

Escribe ! y selecciona la primera opción sugerida.

Guardar todo

CTRL + ALT + s

Pero esto funciona porque lo he configurado. Ir a archivo > preferencias > atajos de teclado y buscar guardar todo. Hay una de las opciones que aparecerá sín atajo asignado con el siguiente código workbench.action.files.saveAll. Editar ese y asignarle el atajo de teclado que se quiera.

Hacer un cambio en todos los archivos

CTRL + SHIFT + f

Seleccionar varias frase en diferentes líneas:

Alt + mouse selecttion.
