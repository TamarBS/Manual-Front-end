Estilos CSS

Hoja de estilos en cascada. Es el lenguaje que aporta el diseño al html.

Sintáxis

selector { propiedad: valor; propiedad: valor; }

Selector

Se utiliza para encontrar el elemento HTML que se quiere afectar con estilos.

Se pueden dividir en 5 categorías:

  • Simple

    Selecciona en base a los nombres identificadores.

    • id: #nombre del id

    • class: .nombre de la clase

    • name: nombre de la etiqueta

    Se pueden utilizar diferentes selectores que afecten a varios nombres identificadores a la vez mediante coma ,

    .clase, #id, nombre-de-etiqueta {... }
    <!-- Los estilos afectarán a la clase, al id y a la etiqueta.-->
  • Combinators

    Seleccionan elementos en función a una relación específica entre ellos.

    • ( space ) descendant: Afecta a todos los hijos del tipo referido.

      Ej.: div p (Afecta a todos los p hijos, nietos, etc que contenga el div)

    • > child: Afecta a los hijos directos que contiene el elemento.

      Ej.: div > p (Afecta a sus p hijos pero no a sus p nietos)

    • + next sibling: Afecta sólo al elemento referido que va inmediatamente después del elemento seleccionado.

      Ej.: div + p (Afecta a su p hermano sólo si está situado inmediatamente después. Si hay otro elemento entre medias no le afectará)

    • ~ subsequent-sibling: Afecta a los elementos referidos que van después de él y se consideran hermanos. Sólo los que van después y no antes. Se diferencia del sibling porque no tienen que ser los inmediatamente siguientes, en este caso sí pueden haber otros elementos entre medias.

      Ej.: div ~ p (Afecta a todos los p hermanos del div que se coloquen por debajo de él)

  • Pseudo-class

    Selectores en base a ciertos estados.

    selector :pseudo-class {
    propiedad: valor;
    }

    Estados de los elementos:

    • :hover : El ratón está pasando por encima del area del elemento.

      ¡IMPORTANTE! El hover tiene que ir después de las pseudo-clases :link y visited, si no, no funciona.

    • :visited : Enlace ya visitado.

    • :link : Por defecto. Enlace sin visitar.

    • :active : El elemento está siento clickado en ese momento. No confundir con focus que es el elemento recibiendo el foco, mediante tab o selección.

      ¡IMPORTANTE! el active tiene que ir después del hover, si no, no funciona.

    • :first-child :afecta al elemento que es el primer hijo de algún otro elemento.

    • :lang

      Permite definir estilos especiales para elementos que contienen la propiedad lang. Es decir, en un texto puedo tener varias palabras en inglés y querer mostrarlas con estilos especiales.

      En esta frase hay terminos en English que se quieren estilar de different manera.

      < style >
      span :lang(en) {
      background-color: cornflowerblue;
      font-weight: bold;
      }
      </ style >

      < p > En esta frase hay terminos en < span lang= "en" > English </ span > que se quier... </ p >

    • focus : El elemento está recibiendo el foco, bien mediante tabulaciones o bien después de hacer click. No confundir con active que es el elemento siento clickado en ese momento.

    Específicos para inputs

    • valid :

      Afecta a los input con un valor válido.

    • invalid :

      Afecta a los input y fieldset con valores no validos.

    • required : Afecta a input que tengan el atributo required.

    • optional : Afecta cualquier input, select o textarea que no tenga el atributo required.

  • Pseudo-elements

    Seleccionan una parte del elemento.

    lista de pseudo-elementos

    selector ::pseudo-element {
    propiedad: valor;
    }

    Utilizar :: en lugar de solo uno : es una actualización de CSS3 para destinguir pseudo-clases de pseudo-elementos.

    • ::first-line : afecta la primera línea de un texto. Sólo afecta a los elementos tipo block.

    • ::first-letter : afecta a la primera letra. Sólo funciona con elementos tipo block.

    • ::before : aplica algún estilo delante del contenido del elemento.

    • ::after : aplica algún estilo detrás del contenido del elemento.

    • ::marker : afecta los estilos de los marcadores en las listas (Los marcadores son los rectángulos, puntos o números delante de los li)

    • ::selection

      Afecta los estilos del subrayado que aparece cuando se selecciona un area del texto.

      Subraya con la selección del ratón para ver el estilo de fondo amarillo y texto rojo del ::selected.

    • ::backdrop

      Aplica estilos al fondo que aparece detrás de los cuadros de dialogo, popup, modales, ventanas emergentes, etc. Por defecto el color es un gris con transparencia.

      El fondo de esta ventana emergente está afectada por el pseudo-selector ::backdrop.
  • Por el atributo

    Selecciona elementos basado en su atributo o valor del atributo.

    elementoHTML[atributo] { ... }
    <!-- Aplicará estilos a los elementos que tengan el atributo. -->

    elementoHTML[atributo="valor"] { ... }
    <!-- Aplicará estilos a los elementos que tengan el atributo que contenta el valor. -->

    elementoHTML[atributo~="palabra"] { ... }
    <!-- Aplicará estilos a los elementos que contengan la palabra dentro del valor del atributo. Solo afecta si la palabra está separada por espacios, por ejemplo -flores no funcionará al estar unido al guión. -->

    elementoHTML[atributo|="valor" { ... }
    <!-- Aplicará estilos a los elementos que contengan sólo la palabra exacta dentro del valor del atributo o también funcionando si la palabra está seguida de un guión para unirlo a otro término. -->

    elementoHTML[atributo^="valor" { ... }
    <!-- Aplicará estilos a los elementos que contengan como primer valor del atributo, ese valor definido, que no tiene porqué ser una palabra entera. -->

    elementoHTML[atributo$="valor" { ... }
    <!-- Aplicará estilos a los elementos que contengan como último valor del atributo, ese valor definido, que no tiene porqué ser una palabra entera. -->

    elementoHTML[atributo*="valor" { ... }
    <!-- Aplicará estilos a los elementos que contengan en algún lugar de su valor del atributo, ese valor definido, que no tiene porqué ser una palabra entera. -->

Ejemplos de selectores:

.center {...} Afecta a todos los elementos con clase center.

p.center {...} Afecta a los elementos p que tengan la clase center.

* {} Selector universal. Afecta a todos los Elementos HTML.

.center

h1, h2, h3 {...} Afectan los mismos estilos a los elementos h1, h2 y h3.

BEM

Metodología para nombrar las clases CSS. Las siglas hacen referencia al orden y contenido que seguirá el nombre de la clase elegido.

  • bloque: define el componente principal, por ejemplo card o button.
  • elemento: define una parte específica dentro del bloque. Título, texto...
  • modificador: define un estilo condicional al elemento. Rojo, focus...

Para separar palabras la sintaxis es la siguiente: .bloque__elemento--modificador

card__title
button button--primary
contact-form__label
product-card__button--buy

Los nombres de más de una palabra se separan por un guíon: contact-form
