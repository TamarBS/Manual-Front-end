Lista de Elementos HTML básicos

<!DOCTYPE html>

Determina el tipo de documento que es. Ayuda al navegador a identificar que el archivo es un documento HTML.

<html lang="es-ES"></html>

Todo el contenido del documento se recoge dentro de estas etiquetas.

Con el atributo obligatorio lang declara el idioma - PAÍS de la página web. Es útil para el navegador y los motores de búsqueda. Se puede obviar el código del país.

<head></head>

Espacio reservado para contenido invisible que sirve para hacer declaraciones y enlaces importantes.

<body></body>

Espacio reservado para el contenido visible de la página web.
