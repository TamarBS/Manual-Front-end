Enlaces

Elementos para enlazar con otro texto, un sitio web, una imagen, etc.

a

Elemento enlace. Es buena práctica utilizarlo cuándo la acción que se busca es de navegación entre páginas. Los estilos por defecto son: Sin visitar Activo Visitado. El cursor cambia al hacer hover.

Crear Marcapáginas

Añadir id al elemento al que se quiere llegar. En el enlace se añade ese #id en su href.

Ir al button
< a href = "#button" > Ir al button </ a >

< button id = "button" > Esto es un button </ button >

La página hace scroll hasta el destino si está en la misma página. Si se quiere hacer un marcador a un elemento en otro documento se utiliza otra ruta en el href del enlace.

< a href = "nombredelarchivo.html#button" >
Imagen como enlace
< a href = "#button" > < img src = "rutaalaimagen" > </ a

Atributos

href

Especifica el destino del enlace.

  • mailto: Se utiliza para dirigir a un email.

    < a href = "mailto:correo@ejemplo.com" > Enviar correo </ a >

target

Indica la forma en la que se abrirá el enlace. Hay que entenderlo como la posibilidad de que existan varios iframes dentro de otros.

_self

Pisa la página en la que estabas. La misma página que ocupaba.

_blank

Abre una nueva ventana.

_parent

Pisa la ventana que contiene la página en la que se encuentra el enlace.

_top

Se abre en la página con mayor jerarquía de todas algo así como el primer padre.

seleccionable

Se puede indicar en el target el nombre del iframe en el que quieres que se abra el enlace.

button

Trae estilos de botón por defecto. Junto con código JS inyectado funciona como un enlace, pero sólo no hace nada. Es buena práctica utilizarlo en ocasiones en las que se ejecute una acción que no sea propia de navegar entre páginas.
