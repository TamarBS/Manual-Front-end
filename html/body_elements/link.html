Enlaces

Elementos para enlazar con otro texto, un sitio web, una imagen, etc.

a

Elemento enlace. Es buena práctica utilizarlo cuándo la acción que se busca es de navegación entre páginas. Los estilos por defecto son: Sin visitar Activo Visitado. El cursor cambia al hacer hover.

Crear Marcapáginas

Añadir id al elemento al que se quiere llegar. En el enlace se añade ese #id en su href.

Ir al button
< a href = "#button" > Ir al button </ a >

< button id = "button" > Esto es un button </ button >

La página hace scroll hasta el destino si está en la misma página. Si se quiere hacer un marcador a un elemento en otro documento se utiliza otra ruta en el href del enlace.

< a href = "nombredelarchivo.html#button" >

Imagen como enlace
< a href = "#button" > < img src = "rutaalaimagen" > </ a

Atributos

href

Especifica el destino del enlace.

  • mailto: Se utiliza para dirigir a un email.

    < a href = "mailto:correo@ejemplo.com" > Enviar correo </ a >

  • tel: Se utiliza para dirigir a una llamada de teléfono.

    < a href = "tel:+34666666666" > Llamar </ a >

target

Indica la forma en la que se abrirá el enlace. Hay que entenderlo como la posibilidad de que existan varios iframes dentro de otros.

  • _self: Pisa la página en la que estabas. La misma página que ocupaba.

  • _blank: Abre una nueva ventana.

  • _parent: Pisa la ventana que contiene la página en la que se encuentra el enlace.

  • _top: Se abre en la página con mayor jerarquía de todas algo así como el primer padre.

  • seleccionable: Se puede indicar en el target el nombre del iframe en el que quieres que se abra el enlace.

Selectores CSS

Estados de los enlaces

  • a:link estado del enlace sin visitar. Por defecto en azul.

  • a:visited enlace ya visitado. Por defecto en morado.

  • a:hover cuándo el ratón está sobre el enlace.

  • a:active justo en el momento en el que es clickado. Por defecto en rojo.

Lista de cursores

Para cada enlace es interesante elegir el tipo de cursor más adecuado que encaje mejor con el contenido y acción del enlace.

  • auto text: cursor para texto.

  • crosshair: cruz negra.

  • default: flecha blanca.

  • e-resize w-resize: flecha horizontal blanca con dos puntas a cada lado.

  • help: flecha blanca y signo de interrogación.

  • move: cruz de flechas blancas.

  • n-resize s-resize: flecha vertical de dos puntas norte sur.

  • ne-resize sw-resize: flecha con dos puntas apuntando noreste y suroeste.

  • nw-resize se-resize: flecha con dos puntas apuntando nuroeste y sudeste.

  • pointer: mano con indice señalando.

  • progress: flecha y círculo de cargando.

  • wait: círculo de cargando.

button

Trae estilos de botón por defecto. Junto con código JS inyectado funciona como un enlace, pero sólo no hace nada. Es buena práctica utilizarlo en ocasiones en las que se ejecute una acción que no sea propia de navegar entre páginas.
