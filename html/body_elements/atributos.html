Atributos de los elementos HTML

Algunos de los atributos más utilizados

class

Diferencia mayúsculas de minúsculas. Nunca comenzar con número ni contener espacios. Dar a varios elementos los mismos estilos. (Punto ".") siempre delante del nombre de la clase.

class=".clase1 .clase2 .clase3"

id

Diferencia mayúsculas de minúsculas. Nunca comenzar con número ni contener espacios. Sólo un único id por elemento. Nombre id único en todo el proyecto. '#' siempre delante del nombre del id.

id="#id"

title

Sirve como tooltip.

Posa un rato el ratón

< p title = "aquí ejemplo de un tooltip" > Posa un rato el ratón </ p >

href

Hyperlink reference

src

Source

Width y height

Ancho y alto. Valor como número sin px porque se da por supuesto

alt

Alternative text. Aparece cuándo el elemento no carga bien o en caso de lectores para ciegos.

style

Para hacer cambios de estilo sólo en ese elemento

El valor de este atributo es diferente:

<tagname style="property:value; property:value; ">
