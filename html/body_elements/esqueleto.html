Elementos HTML para ordenar y agrupar contenido

Los contenidos de una web se pueden agrupar en etiquetas contenedoras que ayudan a mejorar la accesibilida, el orden y la comprensión del código.

Elementos HTML para el esqueleto base

Las webs comparten una estructura de organiación de contenidos basada en varios Elementos HTML que facilitan el orden y la familiaridad.

header

Cabecera del documento o sección. Suele contener, logo, autoría y un elemento de título h1 - h6.

nav

Menú de navegación del sitio web. Cuando hay varios es buena práctica de accesibilidad añadirles un atributo aria-label que los distinga en función del contenido.

section

Separador de contenido sobre un mismo tema, normalmente con un título y un cuerpo de texto. No está relacionado con el contenido principal. Ej: Capítulos, introducción, contacto, noticias.

article

Contenido independiente del resto de contenidos de la web con sentido por sí mismo. EJ: Blog posts, Comentarios, artículos de periódicos.

aside

Anticuado. Algo así como una barra de navegación, espacio para anotaciones o contenido lateral vertical de lo que sea. Está relacionado con el contenido principal.

footer

Pie para información adicional y normas del sitio, arbol web, info legal, etc.

Otros Elementos HTML para declarar contenidos

Algunos de estos elementos sirven para agrupar información dentro de un mismo contenedor, otros son etiquetas para declarar un tipo concreto de información.

details

Sección de contenido desplegable. Debe contener la etiqueta summaryque será el título. Por defecto aparece el contenido oculto. Aparece una flecha delante del "summary".

main

Informa de que es el contenido principal de la página. Sólo debe haber uno por documento.

template

Reservado para contenido que en un principio aparece oculto y se muestra en pantalla cuándo el usuario activa el Javascript que lo hace aparecer.

dialog

Ventana emergente. Atributo "open"

Ventana emergente
div

Elemento block (ocupa todo el ancho por defecto). Se utiliza generalmente como contenedor para organizar otros elementos y aplicar estilos de forma más concreta.

time

Declara que su contenido es una fecha/hora.

data

Permite dar un atributo "value" con contenido legible para la máquina (por ejemplo numérico) y un contenido comprensible para las personas.
