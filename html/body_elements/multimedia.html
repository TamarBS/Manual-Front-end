Multimedia

Elementos que permiten insertar o enlazar con imágenes, vídeos, audio u otras páginas web.

audio

Igual que el vídeo, permite atributos "controls", "muted" y "autoplay" y dentro van los source con varios formatos. El texto dentro de esta etiqueta sólo se ve si el navegador no carga los audios.

canvas

Espacio para crear gráfico mediante JavaScript. Tiene que tener un id para enlazar con el javascript y tamaño de alto y ancho para que no tiemble la página al cargar. Permite crear rectángulos, círculos, textos con diseños especiales, añadir imágenes, etc.

figure

Contenedor para imágenes, ilustraciones, diagramas, fotos, etc. con pie de foto. El elemento "figcaption" sirve como pie de foto y tiene que ir dentro del figure.

< figure >
< img src= "url-imagen" alt= "texto-alternativo" width= "xxx" >
< figcaption > Pie de foto </ figcaption >
</ figure >

i

icon. Etiqueta para añadir icono desde una librería a través del class.

También funcionaría colocando la clase en cualquier elemento inline, por ejemplo span. Pero i es más específico para iconos.

< head >
< script src= "https://kit.fontawesome.com/a076d05399.js" crossorigin= "anonymous" > </ script >
</ head >
< body >
< i class= "fas fa-cloud" > </ i >
< i class= "fas fa-heart" > </ i >
< i class= "fas fa-car" > </ i >
< body >

Librerías de iconos populares

Font awesome Bootstrap Google

iframes

Espacios dónde se pueden incluir una página web dentro de otra.

Para evitar saltos al cargar la página: dar valores width, height.

Abrir página en el iframe

< iframe src= " " name= "iframe_name" title= "..." style= "width:100%" > </ iframe >
< a href= "url al sitio" target= "iframe_name" > Abrir página en el iframe </ a >

¿Cómo reproducir vídeos de youtube?

En un elemento iframe añadir los atributos de alto y ancho y en el "src" poner "https:/www.youtube.com" + "/embed" (de embedded, inyectado) + "/el-id-del-vídeo(buscarlo en el url)"

< iframe width = "315" height = "315" src = "https://www.youtube.com/embed/tgbNymZ7vqY" > </ iframe >

Después del id del vídeo se pueden añadir otras opciones como:

  • autoplay=1: que se reproduzca al cargar.

  • mute=1: que se reproduzca sin sonido.

  • loop=1: para que se reproduzca en bucle.

  • controls=0: para que no se vean los controles.

Se añade un símbolo "?" después del id y entre valores un &.

< iframe width = "315" height = "315" src = "https://www.youtube.com/embed/tgbNymZ7vqY?autoplay=1&mute=1" > </ iframe >

Se pueden reproducir listas de vídeos colocando en el src varias url separadas por coma. ","

img

Espacio reservado para contener una imagen. Darle siempre valores width y height para que al cargar la página no de saltos al cargar el contenido. Los estilos css son más prioritarios que los valores de los atributos. Recomendable usar siempre atributos: alt, height and width.

Acepta formatos animados como '.gif'

< img src = "ruta a la imagen" alt = "Descripción de la imagen" width = "200" >

  • src: Puede ser ruta a un archivo dentro del proyecto o a una url externa.

  • alt: Útil para personas ciegas y cuándo no carga la imagen.

  • width | height: Se lee en píxeles, no hace falta poner 'px'.

map

Crea áreas clickables en una imagen.

< img src= "ruta a la imagen" alt= "Descripción de la imagen" usemap= "#workmap" >

< map name= "workmap" >
< area shape= "..." coords= "..." alt= "..." href= "..." >
< area shape= "..." coords= "..." alt= "..." href= "..." >
</ map >

  • shape: rect | circle | poly | default. Rectángulo, círculo, polígono o toda la imagen.

  • coords: Coordenadas para posicionar las áreas clickables.

    • rect: 4 coords. Esquina superior izquierda = left - top + esquina inferior derecha = left-top.

    • circle: 3 coords. Centro del círculo = left - top + px de radio.

    • poly: Tantas como sean necesarias.

picture

Diferentes imágenes según el tamaño del dispositivo. También para diferentes formatos de imagen a los navegadores, por si no reconoce alguno. Se queda con el primer source que le sirva e ignorará el resto.

< picture >
< source media= "(min-width: 1000px)" srcset= "ruta a la img" width= "xxx" >
< img src= "ruta a la img" > <!-- Imagen por defecto -->
</ picture >

media: condición de tamaño para empezar a aplicar esa imagen.

svg

Scalable vector graphics. Etiqueta que contiene una imagen vectorial svg. Se pueden crear vectores rectángulos, círculos, texto, polígonos y mucho más. Con JavaScript se puede hacer que los objetos sean interactivos.

< svg width = "100" height = "100" >
< circle cx = "50" cy = "50" r = "40" stroke = "green" stroke-width = "4" fill = "yellow" />
</ svg >
< svg width = "400" height = "200" >
< rect x = "10" y = "10" width = "200" height = "100" stroke = "red" stroke-width = "6" fill = "blue" />
</ svg >
< svg width = "300" height = "200" >
< poligon points = "100,10 40,198 190,78 10,78 160,198" style = "fill:lime; stroke:purple; stroke-width:5; fill-rule:evenodd;" />
</ svg >
< svg width = "500" height = "130" >
< defs >
< linearGradient id = "grad1"
< stop offset = "0%" stop-color = "yellow" />
< stop offset = "100%" stop-color = "red" />
< linearGradient />
< defs />
< ellipse cx = "100" cy = "70" rx = "85" ry = "500" fill = "url(#grad1)" />
< text fill = "#fff" font-size = "45" font-family = "verdana" x = "50" y = "86" > SVG </ text >
</ svg >

video

Para incluir un vídeo. Dentro de la etiqueta, el texto que se escriba no se verá, salvo en el caso de que el navegador no pueda reproducir el vídeo. En video se definen los atributos alto y ancho. Dentro de video, van los source con sus atributos "src" y "type" (espeficipar el formato de vídeo) y se puede añadir el elemento "track" que sirve para añadir subtítulos en videos y audios.

Atributos del video

  • controls: Atributo sin valor que añade pause, play y volumen.

  • autoplay: Atributo sin valor para que el vídeo se inicie automáticamente al cargar.

  • muted: Atributo sin valor que va después de autoplay y hará que el vídeo esté sin sonido por defecto
