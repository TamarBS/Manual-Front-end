Gráficas

Elementos HTML que sirven como gráficos para visualizar información.

progress

Crea una barra de progreso. Tiene los atributos "value" y "max". Siempre acompañar de la etiquta label como buena práctica de accesibilidad.

< progress value = "32" max = "100" > 32% </ progress >

meter

Crea una barra de progreso pero no se debe utilizar para conceptos de barra de progreso. Es más bien una gráfica que muestra datos dentro de un rango. Por ejemplo, la cantidad de memoria Ram del ordenador.
