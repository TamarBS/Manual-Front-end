Formatos de texto

Existen diferentes marcados de texto para cambiar el aspecto del texto.

Elementos HTML para formato de texto

b

bold. texto en negrita.

strong

Como la negrita, pero significa palabra importante para los motores de búsqueda.

i

Italics. Cursiva. Se suele utilizar para palabras extrangeras, terminología técnica, el nombre de un barco, etc.

em

Emfasis. Cursiva, pero paa los lectores de texto marcará una pronunciación enfática.

mark

marker. Cómo subrayado por un fosforito.

small

Texto más pequeñito.

del

delete. Texto tachado porque se entiende como eliminado del párrafo.

s

Indica que un texto ya no es correcto, está desactualizado, no es relevante, no es adecuado, etc. Aparece el texto tachado. Es algo así como el del, pero su semántica indica un matíz diferente.

ins

inserted. Texto añadido, algo así como el del, pero en vez de ser rechazado del texto, este es un añadido. Se representa subrayando.

sub

subtext. Texto que se coloca por debajo del texto.

sup

suppertext. Texto que se coloca por encima del texto.

span

Elemento inline que se suele utilizar para aplicar estilos diferentes sólo a una parte del texto.

pre 

preformatted. Texto que
      respeta     los espacios que 
  se añadan   o saltos    de

      línea tal cual se escriben en el código.
      Y con fuente monoscope

code

Presenta el código en monoscope, como pre, pero sin respetar saltos de línea ni espacios en blanco. Se utiliza para mostrar código del ordenador.

kbd

Keyboad. Fuente monoscope y sirve para indicar atajos de teclado.

bdo

bi-dimensional override. Dirección de escritura.

ejemplo de frase con la dirección cambiada.
< bdo dir = "rtl" > ejemplo de fra... < bdo >

rtl = right to left , ltr = left to right

samp

Sample output. Resultado de programa de ordenador. Algo así como la etiqueta que indica que un contenido es el resultado de el cálculo o funcionalidad de la máquina. También se ve como monoscope.

var

variable. Para indicar un contenido que corresponde a una variable. Se muestra en cursiva.

Caracteres reservados

Ej (<, >)

  • &entity_name; EJ.: &lt; =<
  • &#entity_number EJ.: &#60; = <

Otros Elementos poco usados

bdi

bi-directional Isolation. Es útil para otros alfabetos que se escriben de derecha a izquierda y siguen otras normas. Por ejemplo en el alfabeto árabe se colocan en lugares diferentes a los esperados. Para controlar esto se usa el alfabeto entre etiquetas bdi.

Ej sin y con bdi: User إيان: 90 points ≠ User إيان: 90 points

ruby

Normalmente utilizado en japonés, es una etiqueta que contiene un texto que va acompañado de una aclaración sobre cómo se pronuncia.
Los caracteres de la pronunciación van entre etiqutas "rt"

漢 ㄏㄢˋ

Existe otra etiqueta relacionada con esta que es "rp" que sirve para añadir los símbolos de paréntesis alrededor de los caracteres de pronunciación para los casos en los que el navegador no interpreta la etiqueta rt. Así se expresa de otra manera que se trata de la pronunciación del kanji.

u

unarticulated. Es algo así como texto diferente del texto normal. Normalmente texto mal escrito o nombres propios en chino.
