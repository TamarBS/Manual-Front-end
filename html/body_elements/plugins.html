Plug-ins

Permiten expandir las funcionalidades de una web incluyendo servicios como: gps, mapas, escaner de virus, verificaciones de banca, etc.

object

Contenedor para plugins como: lectores de PDF, reproductores de Flash y para incluir Html(por ejemplo una lista que está en otro documento html), incluso imágenes.

< object width = "100%" height = "500px" data = "otro-archivo.html" > </ object >
<-- este object mostraría el contenido de otro archivo html en ese espacio de ancho completo y 500px de alto -->

Puede contener la etiqueta "param" que permite los atributos "name" y "value" para, por ejemplo hacer que un audio se reproduzca en el momento en que se carga.

embed

Embedded object. No tiene etiqueta de cierre. Es anticuado, ya no aparece en HTML5. Sirve igual que el object, como contenedor.
