Dar estilos CSS

list-style-type

Valores

  • upper-roman: el numerador de lista ordenada son número romanos.

  • lower-alpha: el numerador de lista ordenada son letras en minúsculas.

Dar estilos CSS

list-style-type

Valores

  • circle: el marcador de lista ul es un círculo de sólo el borde.

  • square: el marcador de lista ul es un cuadrado solid negro.

Para utilizar una imagen prediseñada como icono de lista.

El valor es la url a la imagen.

list-style-position

Define si la imagen bullet está dentro del cuadro de texto o fuera.
