Elementos HTML para tablas

table

contiene toda la tabla.

caption

Da un título a la tabla que se colocará arriba centrado.

tr

Table row. Contiene al grupo de celdas que forman una fila.

th

Table header. Se usa en lugar del td cuándo el contenido es un encabezado, así es más fácil dar estilos diferentes

td

Table data. Va el contenido. El contenido puede ser cualquier cosa, un enlace, una imagen, otra tabla, etc.

Elementos HTML que agrupan celdas

colgroup

Para seleccionar columnas. Sólo pueden afectar los siguientes estilos: width, visibility, background y border.

  • col: va dentro de un colgroup. Es dónde se define el estilo a aplicar y el número de celdas y dirección afectadas.

thead

Engloba lo que aparezca dentro, generalmente encabezados.

tbody

Engloba lo que aparezca dentro, generalmente contenidos. Es el valor por defecto de todas las tablas ¡cuidado! al dar estilos a esta etiqueta general porque se aplicará en todas las tablas.

tfoot

Engloba lo que aparezca dentro, generalmente el pie.

thead tbody tfoot: Especialmente pensado para ser aplicado en tablas grandes en las que se busca dejar fijos los encabezados y el pie mientras el cuerpo se puede escrolear.

Atributos de tabla

Para elementos colgroup

span: nº de columnas a las que aplicará el estilo definido en el colgroup

Para el resto

class

La clase que se aplicará en las columnas elegidas

colspan

nº de columnas de ancho que ocupara la celda

rowspan

nº de filas de alto que ocupara la celda

scope

No es nada visible. Da información a la máquina sobre que la celda en cuestión es un encabezado. En función de sus posibles valores (col / row / colgroup / rowgroup ) le indicará que los datos a los que encabeza se disponen en columna, en fila, en grupo de columnas, en grupo de filas, o en ambas direcciónes. Ej.: scope="col row"

style

Se puede controlar estilos pero también el ancho y alto de las celdas. Si se añaden valores de porcentaje % se ajustará al valor que tenga su padre.

Propiedades CSS para tablas

Las tablas html no son nada sin estilos css. Estos son los que le aportan claridad y legibilidad. Existen varias propiedades css fundamentales a la hora de diseñar tablas.

border-collapse

Hace que los bordes converjan en uno solo y no quede la tabla con dobles bordes.

table { border-collapse : collapse ; }
border-spacing

Define el espacio que quedará entre las celdas y el contenedor

:nth-child(...) selector par o inpar (even - odd)

Para darles estilo y que la tabla muestre con más claridad las diferentes filas o columnas.
