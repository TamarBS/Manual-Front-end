Elementos HTML propios del Head

La etiqueta head tiene sus propios elementos HTML que no se utilizan en el body.

Ejemplo de un head con bastantes metadatos

< head >
< meta charset = "UTF-8" >
< meta name = "viewport" content = "width=device-width, initial-scale=1.0" >
< meta name = "keywords" content = "HTML, CSS, JavaScript" >
< meta name = "author" content = "Tamar Buelta" >
< meta name = "description" content = "Free Web manuals" >
< meta http-equiv = "refresh" content = "30" >

< link rel = "stylesheet" href = "styles.css" >
< link rel = "icon" type = "image/x-icon" href = "/images/favicon.ico" >

< title > Título del documento </ title >

< base href = "https://www.w3schools.com/" target = "_blank" >

< style >
h1 { color : red ; }
.clase { background-color : blue ; }
</ style >

< script >
function myFunction() {
document. getElementById( "demo"). innerHTML = "Paragraph changed."; } </ script >
< noscript > Lo sentimos, tu navegador no ejecuta JavaScript. </ noscript >

Lista de Elementos HTML metadatos

Los metadatos son datos que aportan datos a la página. Pueden ser datos sobre enlaces con otros documentos, sobre la forma de adaptar el contenido a un dispositivo, sobre la transcripción de caracteres, el modo de apertura de enlaces, etc.

<title></title>

Elemento obligatorio en cada archivo. Título de la página y aparece en la pestaña del navegador. Importante para SEO. Será el texto por defecto en los marcadores y sirve para que los motores de búsqueda almacenen y organicen el mapa de la web.

<link></link>

Enlaza el documento con otros documentos.

  • <link rel="stylesheet" href="styles.css"> Enlaza con una hoja de estilos llamado styles.css

  • <link rel="icon" type="image/x-icon" href="/images/favicon.ico"> Enlaza con un favicon que se verá en la pestaña de la ventana. El favicon mide 16x16 con colores de alto contraste. Se pueden crear favicones online en webs como: favicon.cc. Se aconseja llamar al favicon 'favicon.ico'.

<meta>

Metadata que no será visible en la pantalla pero es útil para navegadores.

  • <meta charset="UTF-8"> el set de caractéres

  • <meta name="keywords" content="HTML, CSS, JavaScript"> define las palabras clave para motores de búsqueda.

  • <meta name="description" content="Free Web tutorials"> una descripción de la web para motores de búsqueda

  • <meta name="author" content="Tamar Buelta"> define el autor del documento

  • <meta http-equiv="refresh" content="30"> que refresque el documento en 30 segundos.

  • <meta name="viewport" content="width=device-width, initial-scale=1.0"> ayuda a que la web se vea bien en todos los dispositivos. Indica que el ancho será el ancho del dispositivo y que se aplicará un zoom adaptado. Sin este meta, la adaptación a móvil y tablet es peor. Mejora el responsive.

<base>

Define la url base y el atributo 'target' que se aplicará a todos los enlaces en el documento.

<script></script>

Permite ejecutar funciones de JavaScript que sólo apectarán a este documento.

<noscript></noscript>: Se añade después del script para mostrar su contenido de texto en caso de que el JavaScript no cargue.

<style></style>

Permite dar estilos CSS que sólo apectarán a este documento.
