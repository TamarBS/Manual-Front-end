Función Math

Permite realizar cálculos para definir las medidas css.

Funciones

  • calc(): hace un calculo utilizando los símbolos + - * /. Puede hacer calculos entre medidas diferentes. Ej.: calc(100% - 100px).

  • max(): Puede contener varios valores separados por una coma ( , ) y lo que hará será elegir la medida del mayor posible.

  • min(): Puede contener varios valores separados por una coma ( , ) y lo que hará será elegir la medida del menor posible.
