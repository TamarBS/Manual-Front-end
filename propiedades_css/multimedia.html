Multimedia

Propiedades CSS que sirven para afectar el aspecto de las imágenes, vídeos, audios, etc.

clip-path

Permite recortar un elementos en formas básicas.

circle(Ø%)

Recorta la imagen en forma de círculo.

Se puede añadir otro valor a la función para situar el centro. Por defecto se coloca en el centro de la imagen, pero se puede colocar en (left, right, top, bottom) creando ilusión de media luna.

ellipse(Ø% Ø%)

Recorta la imagen en forma de elipse.

Igual que en circle se puede situar el centro en las posiciones: left, right, top, bottom creando ilusión de media luna.

polygon(ejeX% ejeY%, ejeX% ejeY%, ejeX% ejeY%, ... )

Recorta la imagen en forma de polígono.

Puede tener tantos parámetros como puntos se quieran añadir al polígono.

shape-outside

Permite adaptar contenidos a una forma clip-path. Por ejemplo por defecto cuando hay una img y un texto el texto se adapta a la imagen de forma rectangular incluso si esta es redonda.

Es algo así cómo recortar, no sólo la imagen, sino su contenedor, permitiendo que lo que tenga alrededor se ajuste mejor a su nueva forma recortada. Su valor es el mismo que el clip-path.

objet-fit

Especifica cómo se ajustan las imágenes y los vídeos al tamaño del contenedor.

object-fit: fill | cover | contain | none | scale-down ;

fill

Es el por defecto. Ocupará el tamaño del contenedor aunque tenga que deformar la imagen.

contain

Conserva sus proporciones y se agranda hasta cubrir el lado más pequeño. Puede quedar espacio del contenedor vacío.

cover

Conserva sus proporciones y se ajusta al lado más grande. Para eso a veces algún contenido de la imagen queda cortado y no se ve.

none

La imagen no se ajusta al contendor. Se muestra en su tamaño original.

scale-down

La imagen se ajusta a su versión más pequeña, ya sea la propia de la imagen o la del lado más corto del contenedor. Es un valor que oscila entre comportarse como none o como contain.

object-position

Se usa conjuntamente con object-fit. En caso de que el contenedor sea más pequeño que la imagen, esta propiedad nos permite decidir dónde se corta. Sería algo así como permitir desplazar la imagen de fondo hasta que en la ventanita se vea justo la parte que queremos ver.

mask-image

Efecto para fusionar dos imágenes. Se llama máscara porque el efecto es el de utilizar la forma de una de las imágenes enmascarándola con el contenido de la otra.

Dos imágenes

Una capa mask-image puede ser un png, un svg, un gradiente, o un elemento svg.Lo importante es que la imagen que aporte la forma deseada tenga transparencia.

< style >
.mask {
mask-image: url(...ruta a la imagen con trasnparencia...);
}
</ style >

< div class= "mask" >
< img src= "...ruta a la imagen jpg..." alt= " ... " width= "Ø" height= "Ø" >
</ div >
Gradientes

El gradiente va de negro a transparente, siendo negro lo más opaco.

< style >
.mask {
mask-image: linear-gradient(color-opaco, transparent);
}
</ style >

< div class= "mask" >
< img src= "...ruta a la imagen jpg..." alt= " ... " width= "Ø" height= "Ø" >
</ div >

Se pueden utilizar los tipos de gradiente "radial-gradient(circle | ellipse)".

SVG Mask element

El elemento HTML <mask> dentro de otro elemento <svg> permiten crear efectos de máscara.

< svg width= "300" height= "200" >
< mask id= "id-del-svg" >
< polygon fill= " color-solid " points= "100 0, 200 200, 0 200" > </ polygon >
</ mask >
< image xlink:href= "ruta-a-la-img" mask= "url(#id-del-svg)" >
</ svg >

Propiedades complemento

mask-clip

Define qué areas son afectadas por la máscara.

border-box | border

Incluye en la máscara el borde y padding.

content-box | content

Sólo incluye el contenido. Sin bordes ni padding.

padding-box | padding

Incle el padding pero no el borde.

fill-box

Es como el content-box pero se define como el cuadro delimitador del objeto.

stroke-box

Es como el border-box pero se define como cuadro delimitador del trazo.

view-box

Utiliza el SVG más cercano como referencia.

Define de qué manera se combinarán las imagenes.

mask-mode

Permite dar un estilo llamado luminance que es algo así como un filtro de imagen con brillo al máximo.

  • match-source: El modo por defecto. Es como alpha, pero para elementos SVG <mask> se comporta de manera diferente.

  • luminance: Aporta ese filtro de brillo al máximo.

  • alpha: Es como el match-source.

mask-origin

Define el punto de origen y colocación de la imagen sobre la máscara. Es una propiedad muy parecida a mask-clip.

Los valores son como los mask-clip: border-box, content-box, padding-box, fill-box, stroke-box y view-box.

mask-position

Define la posición sobre la imagen de la máscara. Por defecto es top-left y se repite horizontalmente y verticalmente.

Los valores pueden ser (top, left, bottom, right), Ø% y puntos.

mask-repeat

Controla la repetición de la máscara sobre la imagen.

Sus posibles valores son: repeat, repeat-x, repeat y, space (la máscara se repite tantas veces como sea posible sin que se corte y el gap se distribuye equitativamente.), round (la máscara se repite y se puede cortar pero se intenta adaptar al tamaño aunque se tenga que distorsionar.) y no-repeat.

mask-size

Controla el tamaño de la máscara.

Los valores pueden ser auto(por defecto el tamaño de la imagen de máscara), px, em, %, etc. contain y cover.
