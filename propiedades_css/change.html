Cambios de apariencia

Propiedades CSS que permiten hacer cambios en otras propiedades del elemento, permitiendo crear transiciones.

transform

Esta propiedad permite mover, rotar, escalar y deformar elementos. Puede contener valores positivos y negativos.

Valores

translate

Desplaza el elemento desde su posición original.

transform: translate(ejex, ejey);

Existen las propiedades translateX y translateY para afectar sólo al eje x o y.

translate(20px, 10px);
rotate

Rota el elemento tantos grados como se especifique.

transform: rotate(ØØdeg);
rotate(-20deg)
scale

Afecta al tamaño del elemento tomando de referencia su tamaño original.

transform: vecesSuX, vecesSuY;

Existen las propiedades scaleX y scaleY para afectar sólo al eje x o y.

scale(0.5, 2)
skew

Deforma el elemento. Su comportamiento es el de inclinar el eje x o y tantos grados se define desde el centro de la línea.

transform: ØØdegX, ØØdegY;

Existen las propiedades skewX y skewY para afectar sólo al eje x o y.

skew(20deg, 10deg)
matrix

Permite aglutinar todos los efectos en una misma propiedad. Aunque admite sólo seis parámetros que no incluyen rotate, porque esto se puede conseguir asignando la misma medida a los skew.

transform: matrix( scaleX, skewY, skewX, scaleY, translateX, translateY);

matrix(1, -0.3, 0, 1, 0, 50)

matrix(1, -0.5, 0.5, 1, 0, 0)

matrix(2, 1, 0.5, 1, 90, 70)

transform-origin

Permite cambiar la posición del centro del efecto.

Por defecto el centro de los efectos tiene valor 50% 50%. Es decir que se "gira", "agranda" o "deforma" tomando como punto de referencia la mitad exacta del elemento.

transform-origin: ejex%, ejey%;
<!-- Tres ejemplos con el mismo rotate -20deg -->
transform-origin: 50% 50%;
transform-origin: 10% 10%;
transform-origin: 80% 90%;
