Cambios de apariencia

Propiedades CSS que permiten hacer cambios en otras propiedades del elemento, permitiendo crear transiciones.

transform

Esta propiedad permite mover, rotar, escalar y deformar elementos. Puede contener valores positivos y negativos.

Funciones transform

translate

Desplaza el elemento desde su posición original.

transform: translate(ejex, ejey);

Existen las funciones transform translateX y translateY para afectar sólo al eje x o y.

translate(20px, 10px);
rotate

Rota el elemento tantos grados como se especifique.

transform: rotate(ØØdeg);
rotate(-20deg)

Existen las funciones transform rotateX, rotateY y rotateZ. En el eje x e y hace un efecto de papel que gira sobre un eje. El z responde como el rotate normal.

rotateX(180deg);
rotateY(180deg);
rotateZ(20deg);

En 90 y 270 grados el rotate x e y no se ve porque sería como ver una hoja de papel de canto.

Propiedad aglutinadora 3D

transform: retate3d(x, y, z, Ødeg);
scale

Afecta al tamaño del elemento tomando de referencia su tamaño original.

transform: vecesSuX, vecesSuY;

Existen las funciones transform scaleX y scaleY para afectar sólo al eje x o y.

scale(0.5, 2)
skew

Deforma el elemento. Su comportamiento es el de inclinar el eje x o y tantos grados se define desde el centro de la línea.

transform: ØØdegX, ØØdegY;

Existen las funciones transform skewX y skewY para afectar sólo al eje x o y.

skew(20deg, 10deg)
matrix

Permite aglutinar todos los efectos en una misma propiedad. Aunque admite sólo seis parámetros que no incluyen rotate, porque esto se puede conseguir asignando la misma medida a los skew.

transform: matrix( scaleX, skewY, skewX, scaleY, translateX, translateY);

matrix(1, -0.3, 0, 1, 0, 50)

matrix(1, -0.5, 0.5, 1, 0, 0)

matrix(2, 1, 0.5, 1, 90, 70)

matrix3d()

Esta funcion permite aglutinar todos los efectos en un estilo 3D.

Otras propiedades relacionadas con trasform

transform-origin

Permite cambiar la posición del centro del efecto.

Por defecto el centro de los efectos tiene valor 50% 50%. Es decir que se "gira", "agranda" o "deforma" tomando como punto de referencia la mitad exacta del elemento.

transform-origin: ejex%, ejey%;
<!-- Tres ejemplos con el mismo rotate -20deg -->
transform-origin: 50% 50%;
transform-origin: 10% 10%;
transform-origin: 80% 90%;

perspective

Define lo lejos que está el elemento del usuario. Es el hijo el que verá aplicados los efectos de perspectiva pero se colocan en el padre.

perspective:none

perspective: 200px

perspective: 100px

perspective-origin

Sólo funciona acompañado de la propiedad perspective.

Define el ángulo desde el que setá viendo el elemento el usuario. Es el hijo el que verá aplicados los efectos de perspectiva pero se colocan en el padre.

perspective-origin: ejeX(left center right length %) ejeY(top center bottom length %);
perspective: 200px, perspective-origin: left top;
Texto dentro del elemento
perspective: 200px. perspective-origin: right bottom;
Texto dentro del elemento
perspective()

También es posible utilizar esta propiedad como una función transform, igual que rotate, scale, etc.

backface-visibility

Define si se muestra o hace desaparecer la cara b de los elementos. La cara que se vería invertida como en un espejo.

Puede tener valor hidden o visible.

resizing
