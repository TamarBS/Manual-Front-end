Contadores CSS

Los contadores son variables CSS que permiten enumerar contenidos. Esto puede ser muy útil en textos largos para poder hacer una buena jerarquía de títulos y subtítulos y facilitar su indexado y posterior búsqueda rápida.

< style >
.code {
counter-reset: section;
}
.counter {
counter-reset: subsection;
}
.counter::before {
counter-increment: section;
content: "Sección" counter( section) ": ";
}
.subcounter::before {
counter-increment: subsection;
content: counter( section) "." counter( subsection) " ";
}
</ style >

Propiedades de los contadores CSS

  • counter-reset: Crea o resetea un contador. El valor es libre y sirve para dar un identificador a ese contador en concreto. Se puede añadir un número detrás del id con el primer valor con el que se quiere trabajar.

  • counter-increment: Define que ese elemento tendrá un contador. Su valor es el nombre que se haya decidido dar en el counter-reset. Por defecto el contador incrementa pero si se quiere hacer que el contador decrezca se añade después del id un número (-1). Así el contador hará un número menos cada vez. Si se quiere incrementar de dos en dos se añade un 2.

  • content: El texto que se añadirá al contador, tipo: Sección 1... Parte 1.3... etc. Se puede añadir el tipo de contador que se quiere después del contenido, por ejemplo upper-roman para usar números romanos.

  • counter() counters() : Añaden el valor del contador dentro del content. counters es más común para lista anidadas que van a tener el mismo tipo de contador.

  • counter-set: permite que el contador empiece en un número concreto.

Es necesario aplicar primero el counter-reset al elemento que contendrá todos los contadores.

Después se definen los elementos que llevarán el contador y se aplica el counter-increment y el content.

Existe la regla @counter-style. Permite crear estilos personalizados para los contadores.
