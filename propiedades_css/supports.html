Regla @supports

Esta regla CSS permite comprobar si una propiedad o valor CSS es compatible con un navegador. En caso de no ser compatible y de que no la identifique, se aplica las propiedades de otro selector.

Sintaxis

@supports (property: value) { /* CSS rules */ }

Se pueden añadir not (para ofrecer alternativas cuando una propiedad no se reconoce) and, or (para condiciones múltiples.)
