Regla @container

Este selector permite definir unas características concretas para un contenedor y darle unas propiedades que sólo les afecten a esos contenedores que cumplen la condición definida.

Sintaxis

.selector {
container-name: containername;
container-type: inline-size;
}
@container containername (containerquery) {
css declarations
}
containerquery puede ser: aspect-ratio, block-size, height, inline-size, orientation, width.
