Transiciones

Las propiedades de transición permiten hacer cambios en las propiedades y que visualmente esos cambios se realicen de forma suave y en un tiempo controlado.

Para crear transiciones en CSS necesitas dos cosas:

  • la propiedad CSS a modificar
  • la duración del efecto
transition: width 2s;

Se pueden añadir más de una transición de propiedad separándolas por comas.

transition: width 2s, height 4s;

Propiedades CSS para transiciones

transition

Especifica la propiedad y el tiempo que le llevará cambiar de estado.

También se puede aplicar la propiedad transform que afectará a la función transform que tenga el elemento.

Es la propiedad aglutinadora de transition-property y transition-duration.

transition: propiedad duración timing-funtion delay

transition-delay

Permite que el efecto tarde en producirse. Los valores son segundos 1s.

transition-duration

Es el tiempo que levará cambiar de un estado a otro.

transition-property

Define qué propiedad es la que se verá afectada por la transición.

transition-timing-function

Especifica la curva de velociad del effecto de transición.

Valores

  • ease: es el por defecto. Empieza y termina despacio y en el medio se acelera.

  • linear: la misma velocidad de inicio a fin.

  • ease-in: sólo tiene inicio lento.

  • ease-out: sólo tiene final lento.

  • ease-in-out: Es como ease pero tarda más en arrancar y parar y no se acelera tanto entre medias.

  • cubic-bezier(n,n,n,n): permite crear una curva de transición totalmente libre.
