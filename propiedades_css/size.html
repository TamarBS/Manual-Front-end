Propiedades que afectan los tamaños

Propiedades que afectan el tamaño de los elementos.

Ancho y alto

widht height

width es el ancho, height es el alto.

Estas propiedades no incluyen el padding, los bores o los márgenes. Sólo define el tamaño del contenido del elemento.

Pueden tener valores: auto/px/cm/%/initial/inherit.

max-width, min-width, max-height, min-height

Tamaños máximos y mínimos para altos y anchos.

Pueden tener valores: auto/px/cm/%/initial/inherit.

Tiene prioridad frente a otra propiedad de sólo ancho y alto.

Box model

Es la forma en la que están compuestos todos los Elementos HTML. Todos los elementos se pueden considerar como una especie de caja.

Border

Padding

box-sizing

Esta propiedad afecta la manera en la que se define el ancho y alto de un elemento teniendo en cuenta el tamaño añadido de su padding y su border.

No afecta al margin de los elementos.
