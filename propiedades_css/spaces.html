Espacios

Propiedades CSS que afectan los espacios vacíos y los márgenes de los elementos

margin

Definen margenes fuera de los bordes del elemento.

Las propiedades pueden ser: margin, margin-top/right/bottom/left.

Pueden tener valores: auto/px/pt/cm/%/inherit.

Se admiten valores negativos.

El valor auto situa el elemento en el centro del ancho pero es obligatorio que exista un valor para width que no sea 100%. Elementos inline como img se pueden centrar si se le da un display:block;.

Ejemplo de un div que se coloca centrado con margin auto.

margin collapse: los margenes top y bottom se colapsan. Es decir, s tengo dos elementos, al de encima le pongo margin 70px y al de debajo 20px, la distancia no será 70+20, se quedará en 70.

padding

Definen márgenes dentro de los bordes del elemento entre los límites y el contenido.

Las porpiedades pueden ser: padding, padding-top/right/bottom/left.

Los valores pueden ser: px/pt/cm/%

No se admiten valores negativos.

outline

Es un espacio añadido entre el border y el margin. Sirve como un segundo border para destacar. Es como el margin en cuánto a que no forma parte del tamaño del elemento.

Propiedades del outline

  • outline-style: Igual que border, puede ser: dotted, dashed, solid, double, groove, ridge, inset, outset, none, hidden.

    Ir a la lista de estilos de border-style.

  • outline-color: lo mismo que border, es para elegir el color del borde.

  • outline-width: igual que el border. Es el ancho de la línea y puede tener valores px/pt/cm/em/etc y también valores predeterminados thin, medium y thick.

  • outline-offset: define el espacio vacío entre la línea del outline y del border. Este espacio no contiene color ni estilos, es siempre transparente.

Propiedad aglutinadora

outline: puede contener: outline-width, outline-style y outline-color. El orden no importa.
