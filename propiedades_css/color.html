Propiedades para editar colores

Lista de propiedades CSS que cambian el color de alguna de las partes de un elementos HTML

opacity

Se puede usar valores entre 0.0 ~ 1.0. 1 la mayor opacidad, 0 transparente.

Esta propiedad afecta a todos los hijos del elemento, así que hay que tener cuidado a la hora de usarlo. Generalmente será mejor utilizar los colores rgba para añadir un fondo de color con transparencia a un elemento.

transparent

Es una palabra clave. Se utiliza como valor y permite definir un color como transparente.

currentcolor

Es una palabra clave. Se utiliza como valor y permite definir un color como el mismo que ya esté dedinido con anterioridad en ese elemento o en algún ancestro directo.

El texto está azul y el borde hereda su color utilizando currentcolor como valor del border-color.

Gradientes

Permite aplicar transiciones suaves entre varios colores. Existen tres tipos de gradientes:

  • linear

    cambia de color de arriba a abajo, derecha izquierda o diagonalmente.

    background-image: linear-gradient(dirección o ángulo, color1 posición%, color2 posición%, ...);

    Top to bottom es el por defecto. Para añadir los valores de dirección se añade: to + dirección. Ej.: to right, de izquierda a derecha. to bottom right, de arriba a la izquierda a abajo derecha en diagonal.

    Ángulo

    Si se quiere añadir un angulo concreto que no sea los por defecto de top, left, bottom y right en el lugar del valor de dirección se coloca el ángulo elegido segido de "deg". Ej.; 175deg, red, yellow.

    Repetición

    Se pueden repetir los gradientes cambiando el valor linear-gradient() por repeating-linear-gradient().

    Se puede añadir un valor de porcentaje después de cada número para controlar el espacio que ocupan y las veces que se va a repetir el gradiente.

    acepta el valor de dirección también y de ángulos igual que linear-gradient().

    background-image: repeating-linear-gradient(red, yellow 30%, green 20%);

    background-image: repeating-linear-gradient(red, yellow 7%, green 50%);

  • radial

    Cambia de color desde un punto central hasta sus extremos.

    background-image: radial-gradient(forma( ellipse | circle ), tamaño ( closest-side | farthest-side | closest-corner | farthest-corner )at posición, color1 posición%, color2 posición%, ...);
    background-image: radial-gradient(circle, closest-side at center, red, yellow 7%, blue 100% );

    Creo que lo del tamaño se define cogiendo de referencia un linear gradient que va en alguno de estas direcciones:

    • closest-side: El lado más próximo. Genera el gradient en horizontal y vertical desde el centro hasta el borde más cercano y ya con eso genera el aro.

    • farthest-side: El lado más alejado. Genera el gradient en horizontal y vertical desde el centro hasta el borde más alejado y ya con eso genera el aro.

    • closest-corner: Las esquinas más próximas. Genera el gradient desde el centro hasta la esquina más cercana y ya con eso genera el aro.

    • farthest-corner: Las esquinas más alejadas. Genera el gradient desde el centro hasta la esquina más alejada y ya con eso genera el aro.

    Se pueden hacer repeticiones de gradientes utilizando repeating-radial-gradient en lugar de radial-gradient.

  • conic

    Se explica mejor con una imagen.

    background-image: conic-gradient(from [angle] at [position,] color [degree], ...);

    Ejemplos

    background-image: conic-gradient(red 0deg, red 90deg, yellow 90deg, yellow 180deg, green 180deg, green 270deg, blue 270deg); border-radius: 50%;

    background-image: conic-gradient(at 60% 45%, red, yellow, green);

    background-image: conic-gradient(from 90deg, red, yellow, green);

    background-image: repeating-conic-gradient(red 10%, yellow 20%);
