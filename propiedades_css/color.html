Propiedades para editar colores

Lista de propiedades CSS que cambian el color de alguna de las partes de un elementos HTML

opacity

Se puede usar valores entre 0.0 ~ 1.0. 1 la mayor opacidad, 0 transparente.

Esta propiedad afecta a todos los hijos del elemento, así que hay que tener cuidado a la hora de usarlo. Generalmente será mejor utilizar los colores rgba para añadir un fondo con transparencia a un elemento.
