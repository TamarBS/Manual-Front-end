Responsive

Hace referencia a la correcta adaptación del diseño web en los diferentes dispositivos y tamaños desde los que un usuario puede utilizar la página web.

Media queries

Significa "consulta de medios", es decir, las media queries hacen una consulta para obtener información del tamaño de dispositivo, orientación y resolución de pantalla del cliente.

Es una regla CSS que permite: obtener el alto y ancho del visor, la orientación (landscape=horizontal o portrait=vertical) y la resolución de pantalla..

Comúnmente se utiliza para hacer estilos especiales en medidas separadas en los conceptos: desktop=pantalla de ordenador de sobremesa, laptop=pantalla de portátil, tablets=pantalla de tablet y mobile=pantalla de teléfonos inteligentes.

Sintaxis
@media not|only tipo de medio (all | print | screen) and características del medio and características del medio {
...
}
<!-- se pueden añadir nuevas media queries a continuación separándolo por comas -->
@media (max-width:360px) and (min-width:1100px), (min-width:1101px) {
...
}

El tipo de medio es opcional ponerlo. El por defecto es all. Sin embargo si quieres utilizar el not o el only es necesario especificar el tipo de medio.

En función de si la consulta de tamaño de pantalla, orientación o resolución coincide con el dispositivo, la media query devolverá true o false. Cuándo el resultado de la consulta es true, todos los estilos dentro de la media query son ciertos y por lo tanto se aplican. Si el resultado es falso esos estilos permanecen inactivos.

Tipos de medio

Dependiendo del medio por el que se mostrará el proyecto puede ser:

  • all: Para los dos tipos de medio. Online y Offline. Virtual y físico.

  • print: Preparado para impresoras. Medio físico de impresión.

  • screen: Preparado para pantallas digitales. Medio virtual.

Características de los medios

Se especifican ciertas características para definir en qué casos se van a aplicar esas reglas media queries.

  • orientation: La orientación landscape o portrait.

  • max-height: Empieza a contar desde arriba a abajo

  • min-height: Empieza a contar desde abajo a arriba.

  • height: Todo el alto de pantalla.

  • max-width: Empieza a contar desde la izquierda hasta la derecha

  • min-width:Empieza a contar desde la derecha hacia la izquierda

  • width: Todo el ancho de pantalla

  • min|max-device-width|height: Define el tamaño del dispositivo no del navegador.

Para trabajar en tamaños pequeñas se usa el max-width.

Para tamaños grandes se utiliza el min-width.

Ejemplo práctico

Estas viéndolo en un smartphone en posición vertical

@media( max-width: 360px) and ( orientation: portrait) { .phone-portrait { display: block; } }

Estás viéndolo en un smartphone en posición horizontal

@media( orientation: landscape) { .phone-landscape { display: block; } }

Estás viéndolo en una tablet o portátil

@media( max-width: 1439px) and ( min-width: 361px) { .laptop { display: block; } }

Estás viéndolo en la pantalla de un ordenador de sobremesa o en una tele.

@media( min-width: 1440px) { .desktop { display: block; } }

Typical breakpoints

Son las medidas standard en las que se suele trabajar con media queries por ser los tamaños de dispositivo más comunes.

  • max-width: 600px; smartphones

  • min-width: 600px; tablets en posición vertical y smartphones grandes

  • min-width:768px; tablets en posición horizontal

  • min-width: 992px; portátiles, pantallas de escritorio.

  • min-width: 1200px; portátiles grandes, pantallas de escritorio

  • ... televisores

viewport

Se denomina viewport al area visible por el usuario de una página web.

HTML5 permite introducir un método para controlar el viewport añadiendo un meta en el head que contiene lo siguiente:

< meta name= "viewport" content="width=device-width, initial-scale=1.0" />

  • width=device-width: Define el ancho de la página web como el ancho del dispositivo.

  • initial-scale=1.0: Define un nivel de zoom para que se vea escalado y más grande en dispositivos pequeños.

Conseguir multimedia responsive

Es posible conseguir que una imagen se ajuste a los diferentes tamaños. Lo más aconsejable es definir un max-width=100%; que hará que se haga más pequeño si es necesario y se hará grande hasta su tamaño original, después no se seguirá ampliando y por tanto no se deformará tampoco. El alto se define en auto para que respete las proporciones de la imagen original.

break-after, break-before y break-inside

Esta propiedad especifica dónde colocar saltos de página para aquellos casos en los que se quiera controlar el modo de impresión.
