JavaScript

Es el lenguaje de programación web más popular. Puede calcular, manipular y validar datos. Fue inventado por Brendan Eich en 1995. En 1997 se convirtió en un estándar ECMA-262. ECMAScript es el nombre oficial de este lenguaje.

Sintaxis

El punto y coma separa estados. Hay que añadirlo al final de cada estado. En algunos casos no es obligatorio pero siempre es recomendable ponerlo.

Se ignoran los espacios, pero se recomiendan añadirlos para mejor comprensión del código. Poner es pacio entre operadores es una buena práctica ( = + - * / )

Se aconseja no hacer líneas de más de 80 caracteres y pasar a la siguiente línea.Cuándo una línea se pasa de caracteres se recomienda cortarla por alguno de sus operadores.

Conceptos básicos

Está formado por funciones que contienen métodos y eventos que son los que arrancan las funciones.

Sí se distingue mayúsculas de minúsculas.

No se permite usar - guión como separador de palabras en nombres de variables.

Para nombrar variables la convención es empezar con una palabra en minúsuclas y seguir la siguiente con la primera en mayúsculas. Ejemplo.: firstName, lastName.

Colocación de JavaScript en documentos HTML

Se coloca dentro de las etiquetas <script> </script>

Puede ir tanto dentro del body como dentro del head. Colocarla al final del body ayuda a la velocidad porque será lo último que cargue de la página.

Lo más recomendable es colocarlos en documentos externos y enlazarlos en el head o el body del documento HTML de la siguiente manera:

<script src="myScript.js"></script>

Métodos HTML

Para encontrar elementos

getElementById()

Permite seleccionar un elemento HTML por su id.

Para añadir contenidos

innerHTML

Sustituye el contenido del Elemento HTML al que apunta. Se usa para cambiar el elemento HTML.

innerText

Sustituye el texto contenido en un Elemento HTML. Se usa para cambiar sólo el texto de un Elemento HTML.

document.write()

Se utiliza por razones de testeo. Este método sobre-escribe todo el HTML.

window.alert()

Crea una ventana emergente.

La palabra clave "window" se puede obviar porque es un objeto de alcance global. Esto significa que las variables, propiedades y métodos por defecto pertenecen al objeto window.

console.log()

Este método inserta texto en la consola del navegador. No se verá en pantalla pero sí en las herramientas de desarrollo. Es útil a la hora de hacer depuración del código (debugging).

Para cambiar propiedades CSS

.style.propiedad

document.getElementById("demo").style.fontSize = "35px";

Para realizar acciones

print()

Permite abrir la ventana de impresión del navegador, generando una página imprimible con los contenidos de la web.

Programas JavaScript

Un programa de ordenador es una lista de instrucciones para ser ejecutadas por un ordenador. En programación estas instrucciones se les llama estados(statements).

En HTML los programas JavaScript se ejecutan por el navegador. Se ejecutan de uno en uno en el orden en el que aparecen en el código.

Estados JavaScript

Se componen de:

  • valores: Hay dos tipos de valores: fijos(literales) y variables. Literales=(números(numbers) y cadenas de texto(strings)). Variables=se utilizan para contener datos. (var, let y const).

    El operador = sirve para asignar un valor a una variable.

  • operadores: ( + - * / ) Aplica operaciones aritméticas. ( = ) se utiliza para asignar valores. Los valores también pueden ser strings no sólo números.

  • expresiones: Es la convinación de valores, variables y operadores.

  • palabras clave: Son con las que se suele empezar. Identifican una acción JavaScript que se quiere llevar a cabo. ( var(variale) let(un bloque variable) const(constante) if(condición) switch(diferentes casos) for(iteracion) function(declara una función) return(devuelve un valor) try(incluye un manipulador de errores) ) Estas son palabras reservadas, lo que significa qu eno se pueden utilizar como nombres de variables.

  • comentarios: Son estados que no se ejecutan. Se utiliza // o /* */ . Son ignorados a la hora de ejecutar estados.

Identifiers/Names

Se utilizan para dar nombre a variables, funciones y palabras clave.

Tienen que cumplir las siguientes normas y comenzar siempre con letras a-z o A-Z, el símbolo $ o _. No puede empezar con números ni níngún otro símbolo que no sea $ o _ .
